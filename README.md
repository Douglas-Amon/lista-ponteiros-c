# 📌 Lista de Exercícios sobre Ponteiros em C

Projeto desenvolvido para a disciplina **DCA3303 - Programação Avançada**.

Este repositório reúne exercícios em linguagem **C** com foco no estudo de **ponteiros**, abordando conceitos fundamentais para manipulação de memória, passagem por referência e estruturas dinâmicas.

---

## 🚀 Conteúdos abordados

* Ponteiros e endereçamento de memória
* Passagem de parâmetros por referência
* Manipulação de vetores com ponteiros
* Manipulação de matrizes com ponteiros
* Alocação dinâmica de memória
* Uso de `malloc`, `calloc`, `realloc` e `free`
* Relação entre arrays e ponteiros
* Funções com ponteiros como parâmetros

## 🧠 Objetivo

O objetivo desta lista é praticar o uso de ponteiros em C, desenvolvendo maior compreensão sobre:

* acesso indireto a variáveis
* manipulação eficiente de memória
* construção de funções mais flexíveis
* uso correto de memória dinâmica
